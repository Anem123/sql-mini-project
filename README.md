# sql-mini-project
Beginner SQL mini project with dataset and solutions

##  Overview
This project is designed for beginners to practice SQL concepts using a simple sales dataset.

##  Concepts Covered
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- Aggregate Functions

##  Files
- dataset.sql → Table creation & data
- sales_project.sql → Practice queries & solutions
- sql_mini_project.pdf → Full project (questions + answers)

##  Sample Questions
1. Total revenue per day  
2. Top-selling products  
3. Revenue > 150 days  

##  How to Run
1. Create table and Insert data using dataset.sql    
2. Run queries from sales_project.sql  

##  Goal
To build strong SQL fundamentals for Data Engineering roles.
