# SQL Mini Project - Sales Analysis

## 📌 Overview
This is a beginner-friendly SQL project designed to practice core SQL concepts using a sales dataset.

## 🧠 Skills Demonstrated
- Data querying using SQL
- Aggregations (SUM, AVG)
- Filtering (WHERE, HAVING)
- Data grouping (GROUP BY)
- Sorting results (ORDER BY)

## 📂 Project Structure
- dataset.sql → Table creation and data insertion
- sales_project.sql → Practice questions and solutions
- sql_mini_project.pdf → Full project explanation

## 📊 Business Questions Solved
- Total revenue per day
- Top-selling products
- Revenue analysis by product
- High revenue days identification

## 🚀 How to Use
1. Run dataset.sql to create table and insert data
2. Run queries from sales_project.sql
3. Compare your answers with solution section

## 🎯 Goal
To build strong SQL fundamentals for Data Engineering and Analytics roles.
